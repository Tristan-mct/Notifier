Installer le package

	Recupérer son Token Slack sur :

	https://api.slack.com/legacy/custom-integrations/legacy-tokens

	Au milieu de la page, cliquer sur 'Create Token'

	##########

	Puis naviguer jusqu'au dossier du package :

	> cd ~/Downloads/MDM_slack

	Lancer l'installateur :

	> ./install

	Entrer le token utilisateur quand celui-ci est demandé.

	Le package s'installe tout seul dans le site-package de l'environnement actif.

	##########

	Le packa s'utilise comment dans le fichier Demo.py
